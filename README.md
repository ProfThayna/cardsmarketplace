# Projeto Marketplace de cartas desenvolvido para teste usando Vue3 e Tailwind

## Requirements

- Vue3
- npm (Node Package Manager)
- Tailwind CSS
- Axios
- Vue Router

Follow these steps to set up the server:

1. **Clone the repository**
```
   git clone
```

2. **Install Dependencies**
```
   npm install
```

3. **Start the Server**
```
   npm run serve
```
